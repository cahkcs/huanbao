## 🚀 部署流程（宝塔面板 5 步搞定）

### 第 1 步：创建 MySQL 数据库

```
宝塔面板 → 数据库 → 添加数据库
```

填写以下信息：

| 字段 | 建议值 | 说明 |
|------|--------|------|
| 数据库名 | `insect_eco` | 可自定义，记住它 |
| 用户名 | `insect_eco` | 自动生成或自定义 |
| 密码 | 点"生成"按钮 | ⚠️ **复制保存好！** |
| 访问权限 | 本地服务器 | ✅ |
| 字符集 | `utf8mb4` | ✅ |

点击 **提交**，记录下数据库名、用户名、密码。

---

### 第 2 步：修改配置文件

编辑 **`api/config.php`**，修改以下 **5 个值**（用 👈 标注的地方）：

```php
'database' => [
    'type'      => 'mysql',
    'host'      => 'localhost',        // ✅ 通常不用改
    'port'      => '3306',             // ✅ 通常不用改
    'database'  => 'insect_eco',       // 👈 改成你的数据库名
    'username'  => 'insect_eco',       // 👈 改成你的数据库用户名
    'password'  => '你的MySQL密码',    // 👈 改成你的数据库密码
],

'admin' => [
    'username' => 'admin',             // ✅ 管理员用户名
    'password' => '你的后台密码',      // 👈 设置后台登录密码
],
```

> 💡 **示例**：如果你在宝塔创建的数据库信息是：
> - 数据库名：`insect_eco_user`  
> - 用户名：`insect_eco_user`  
> - 密码：`AbCdEf123456`  
> - 想设置后台密码：`MyAdmin@2024`
>
> 那么 config.php 中填：
> ```php
> 'database'  => 'insect_eco_user',
> 'username'  => 'insect_eco_user',
> 'password'  => 'AbCdEf123456',
> // ...
> 'admin' => [
>     'password' => 'MyAdmin@2024',
> ],
> ```

保存文件 ✅

### 第 3 步：上传到服务器

#### 打包上传

```bash
# 在 huanbao 目录的上一层执行
# 将整个 huanbao 文件夹打包
zip -r huanbao.zip huanbao/
```

#### 宝塔面板操作

```
宝塔面板 → 文件 → 进入 /www/wwwroot/
→ 新建目录：你的域名（如 insect.mitan.icu）
→ 进入此目录
→ 点击"上传" → 选择 huanbao.zip
→ 上传完成后右键 → 解压
```

#### 设置权限

```
在 /www/wwwroot/你的域名/ 目录下：
→ 全选所有文件
→ 点击"权限"
→ 设置为 755
→ 确定
```

## 🔐 管理员密码管理

### 初始密码

- **用户名**: `admin`
- **初始密码**: 你在 `api/config.php` 中设置的 `admin.password`
- **存储位置**: MySQL 数据库的 `admin_users` 表

### 修改密码（两种方式）

**方式一：通过后台界面（推荐）**
```
1. 登录后台：https://你的域名/cahkcs/index.html
2. 找到 "🔑 修改管理员密码" 面板
3. 输入旧密码 + 新密码 + 确认新密码
4. 点击 "🔐 修改密码"
5. 2 秒后自动退出，请使用新密码重新登录
```

**方式二：通过 phpMyAdmin 直接修改**
```
1. 宝塔面板 → phpMyAdmin
2. 选择你的数据库（如 insect_eco）
3. 打开 admin_users 表
4. 编辑 password 字段，改为新密码
5. 点击"执行"保存
```

## 💾 数据表结构

系统首次运行时会 **自动创建** 以下表：

### `admin_users` - 管理员账户
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT | 主键，自增 |
| username | VARCHAR(50) | 用户名（默认: admin） |
| password | VARCHAR(255) | 密码（存储在数据库中） |

| updated_at | DATETIME | 更新时间 |

### `messages` - 留言
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT | 主键 |
| nickname | VARCHAR(50) | 昵称 |
| content | TEXT | 留言内容 |

| ip | VARCHAR(45) | IP地址 |
| user_agent | VARCHAR(200) | 浏览器信息 |
| created_at | DATETIME | 创建时间 |

| 字段 | 类型 | 说明 |
|------|------|------|
| key | VARCHAR(50) | 键名（visit_count / admin_password） |

## ❌ 常见问题

### Q1：访问 API 返回空白或 500

**原因**：MySQL 连接配置错误

**解决方法**：
```
1. 检查 api/config.php 中的 5 个数据库配置项
2. 确认 MySQL 服务正在运行
   （宝塔面板 → 软件 → MySQL → 状态）
3. 查看 PHP 错误日志
   （宝塔面板 → 网站 → 你的域名 → 日志）
```

### Q2："数据库连接失败"

**原因**：数据库名、用户名或密码不正确

**解决方法**：
```php
// 在 config.php 中再三确认这 3 个值
'database'  => '你的数据库名',   // 必须和宝塔创建的一致
'username'  => '你的用户名',     // 注意区分大小写
'password'  => '你的密码',       // 注意特殊字符
```

### Q3："数据表初始化失败"

**原因**：自动建表失败，权限不足

**解决方法**：通过 phpMyAdmin 手动导入
```
1. 宝塔面板 → 数据库 → 管理（phpMyAdmin）
2. 点击你的数据库
3. 点击"SQL"选项卡
4. 点击"选择文件" → 选择项目中的 api/init_database.sql
5. 点击"执行"
```

### Q4：管理后台登录提示"请求失败"

**原因**：`api/` 文件夹中的 PHP 文件没上传

**解决方法**：
```bash
# 确认服务器上有这 3 个文件
/www/wwwroot/你的域名/api/config.php    ✅
/www/wwwroot/你的域名/api/Database.php   ✅
/www/wwwroot/你的域名/api/index.php      ✅
```

### Q5：网站图标（favicon）不显示

**原因**：浏览器缓存

**解决方法**：
```
1. 强制刷新：Ctrl + F5（Windows）或 Cmd + Shift + R（Mac）
2. 清除浏览器缓存
3. 确认 ing/icu.png 文件存在
```
