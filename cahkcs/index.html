📌 数据库配置说明

当前系统使用 localStorage 作为数据存储（模拟数据库）

如需连接真实数据库，请修改 JavaScript 中的以下函数：

• getMessages() - 获取留言数据 → 改为从 API/数据库查询

• saveMessages(messages) - 保存留言数据 → 改为写入 API/数据库

• getVisitCount() - 获取访问量 → 改为从数据库读取

• saveVisitCount(count) - 保存访问量 → 改为写入数据库

• ADMIN_PASSWORD = 'admin123' - 管理员密码 → 改为从数据库验证
